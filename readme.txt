                                                                                        学生成绩管理系统
项目简介
学生成绩管理系统是一个基于 Python 开发的管理系统，旨在帮助学校或教育机构方便地管理学生的成绩和课程信息。系统支持学生信息的录入、成绩管理、课程安排、数据导入导出等功能。通过该系统，用户可以有效地管理学生的成绩数据并生成相关报表。

功能介绍
学生管理：添加、删除、更新学生信息，包括姓名、学号、班级等基本资料。
课程管理：管理课程信息，支持添加、删除课程及其属性。
成绩管理：录入和更新学生的课程成绩，自动计算总成绩和平均成绩。
数据导入导出：支持从 CSV 文件批量导入学生、课程和成绩数据，支持数据的导出以备份和分享。
用户界面：提供简单的命令行界面，或者基于 ui.py 的图形界面供用户使用（根据项目需求）。

文件结构
main.py：主程序文件，运行该文件启动系统。
database.py：数据库连接和操作的模块，定义了数据表的操作方法。
data_export_import.py：实现数据的导入和导出功能，支持 CSV 文件格式。
settings.py：系统配置文件，包括数据库连接设置等。
db_setup.sql：数据库初始化脚本，创建所需的数据表。
data_insert.sql：用于插入测试数据的 SQL 文件。
students.csv、courses.csv、scores.csv：示例数据文件，用于导入学生、课程和成绩信息。
ui.py：用户界面代码，提供交互功能（若为图形界面）。
系统使用手册说明.pdf：系统的使用手册，介绍了各功能模块的操作方法。
系统设计描述文档.pdf：系统设计的详细说明，包括模块划分和系统架构。
项目演示视频.mp4：项目的操作演示视频。

常见问题
数据库连接错误：请确保在 settings.py 中正确配置了数据库连接信息。
数据导入失败：检查 CSV 文件格式是否正确，确保每列与系统要求的字段对应。
无法启动用户界面：如果使用 ui.py 作为图形界面，请确保已安装相应的图形库（如 Tkinter 或 PyQt）。
